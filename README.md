### Prerequisites

`python2.7`, `pip`, `virtualenv`.

Install them on ubuntu-like system with `apt-get install python2.7 virtualenv`

### Dependencies

Install all pip dependencies with `pip install -r reqs.txt` (make sure you are
in the virtualenv)

### Datasets
